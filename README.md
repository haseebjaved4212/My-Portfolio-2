# Haseeb Javed's Portfolio

A modern, interactive, and fully responsive personal portfolio built with React, Vite, and Tailwind CSS. The portfolio features a sleek design, smooth animations using Framer Motion, and a unique interactive terminal emulator to engage visitors.

##  Features

- **Hero Section**: An engaging introduction with smooth entrance animations.
- **About Section**: A concise biography and background information.
- **Skills Section**: An organized display of technical skills (Frontend, Backend, Databases, etc.).
- **Projects Section**: Highlights of recent work and side projects.
- **Contact Section**: Links to get in touch via email or social platforms.
- **Interactive Terminal**: A unique hidden feature! Open the terminal to execute commands like `help`, `about`, `skills`, `projects`, `contact`, `whoami`, and even download the `resume`.
- **Theme Support**: Includes a ThemeProvider for managing application themes.
- **Smooth Animations**: Powered by Framer Motion.

##  Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)

##  Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) (v18 or higher recommended) and `npm` installed.

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd My-Portfolio-2
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

### Running Locally

Start the development server:

```bash
npm run dev
```

The app will be available at `http://localhost:3000` (or whichever port Vite uses).

### Building for Production

To build the app for production, run:

```bash
npm run build
```

You can preview the built production bundle using:

```bash
npm run preview
```

## 💻 Terminal Commands

The integrated interactive terminal supports the following commands:
- `help` - Lists available commands
- `about` - Short bio
- `skills` - Lists tech stack
- `projects` - Lists projects
- `contact` - Shows contact info
- `resume` - Triggers resume download
- `whoami` - Short one-liner
- `clear` - Clears the terminal output
